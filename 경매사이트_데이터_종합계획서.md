## 3부. 외부 API 연동 추가 데이터 (신규 개발 필요)

### 3-1. 실거래가 연동 ⭐ (최우선)

| 항목 | 내용 |
|------|------|
| API | 국토교통부 실거래가 공개시스템 |
| 비용 | **무료** (공공데이터포털 API 키 필요) |

## 4부. 자체 계산/파생 데이터 (외부 API 불필요)

기존 DB 데이터만으로 즉시 생성 가능한 고부가가치 데이터입니다.

### 4-1. 수익률 시뮬레이션 엔진

```
auction_profitability (신규 또는 실시간 계산)
├── internal_id
├── estimated_sale_price   — 예상 낙찰가 (과거 통계 기반)
├── acquisition_tax        — 취득세 (감정가 × 4.6%)
├── registration_tax       — 등록세
├── broker_fee             — 중개보수
├── tenant_deposit_risk    — 인수 보증금 합계
├── total_investment       — 총 투자금
├── expected_market_price  — 예상 시세 (실거래가 기반)
├── expected_profit        — 예상 수익
├── expected_roi           — 예상 수익률 (%)
└── rental_yield           — 임대 수익률 (%)
```

### 4-2. 낙찰 예측 모델

| 지표 | 계산 방법 |
|------|-----------|
| 예상 낙찰가 범위 | 동일 법원+물건종류+유찰횟수별 과거 낙찰가율 평균/중위값 |
| 유찰 확률 | 동일 조건 물건의 과거 유찰률 |
| 경쟁도 | 같은 기일 같은 법원 물건 수 대비 평균 입찰자 수 |

### 4-3. 물건 난이도 등급

```
등급 산정 기준:
A등급 (초보 추천) : risk_score ≤ 10, 임차인 0명, 소멸기준 이후 권리만 존재
B등급 (보통)     : risk_score ≤ 25, 임차인 1명 이하
C등급 (주의)     : risk_score ≤ 50, 대항력 임차인 존재
D등급 (고난도)   : risk_score > 50, 유치권/법정지상권/미등기 등
```

### 4-4. 권리분석 자동 요약

| 항목 | 산출 |
|------|------|
| 말소기준권리 | auction_registry에서 inherit_type='소멸기준'인 행 |
| 인수 권리 목록 | inherit_type='인수'인 행 자동 추출 |
| 선순위 임차인 여부 | 전입일 < 말소기준일인 임차인 자동 감지 |
| 예상 인수 금액 합계 | 인수 권리 금액 + 대항력 임차인 보증금 합산 |

## 5부. AI 기반 자동 콘텐츠 생성

### 5-1. 물건별 투자 브리핑 (LLM 자동 생성)

```
입력: 감정평가서 + 임차인 + 등기부 + 실거래가
출력: 
  "부산 연제구 거제한양 아파트 10층 — 감정가 7.69억, 최저가 7.69억(100%).
   대항력 있는 임차인 1명(보증금 2.5억). 최근 6개월 동일 단지 실거래가 평균 9.2억.
   예상 수익률 12~18%. 다만 대항력 임차인 명도 리스크 존재.
   ★ 난이도 C등급 — 전문가 상담 권장"
```

### 5-2. 블로그 자동 포스팅 연동

기존 티스토리 자동화 시스템과 연계하여:
- 매일 신규/변경 물건 중 조회수 상위 10건 → AI 분석글 자동 생성 → 티스토리 발행
- 컨설팅 CTA 버튼 자동 삽입

### 5-3. 위험도 해설문

```
risk_keywords에 따른 자동 해설:
- ["유치권"] → "유치권 주장이 있는 물건입니다. 점유자가 공사대금 등을 이유로..."
- ["법정지상권"] → "토지와 건물 소유자가 달라 법정지상권이 성립할 수 있습니다..."
- ["대항력"] → "전입일자가 말소기준권리보다 앞서는 임차인이 존재합니다..."
```
